계정 삭제 방법

앱 사용 중 언제든지 아래 방법 중 하나로 계정 및 관련 데이터를 삭제할 수 있습니다:

  1. 앱 내 메뉴: 설정 > 계정 삭제 메뉴를 통해 직접 삭제할 수 있습니다.
  2. 이메일 요청: 아래 이메일로 요청해 주세요.
    이메일: support@unidev.co.kr
    제목: [체중메모] 계정 삭제 요청
    내용에 가입 이메일 또는 사용자 ID를 포함해 주세요.

📂 삭제 또는 보관되는 데이터

데이터 항목 삭제 여부 보관 기간
